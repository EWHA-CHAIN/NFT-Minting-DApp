## 실행

example 폴더의 환경을 사용하는 경우, 반드시 Marketplace.js와 util/interact.js에 있는 contractAddress 변수에 Contract 주소를 입력할 것

터미널에서 해당 파일 경로로 이동 후 `npm install` 실행

그 후 `npm start` 를 입력 후에 http://localhost:3000 으로 접속해서 실행하는지 확인
